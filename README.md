# ビザ申請サポートシステム デモ

日本語学校向けの在留期間更新申請サポートシステムのデモアプリケーションです。

## 🚀 デモサイト

[https://visa2-demo.netlify.app](https://visa2-demo.netlify.app)

## 📋 機能

### 学生一覧ページ
- 学生の申請状況を一覧表示
- 申請締切日と残り日数の色分け表示
- 提出書類数の可視化
- ステータス別フィルタリング
- レスポンシブデザイン対応

### 学生詳細ページ
- 学生の基本情報表示
- 書類ごとの承認・差戻し機能
- リアルタイムステータス更新
- 書類プレビュー機能（デモ）
- 申請実行機能

### 主要機能
- **ステータス自動管理**: 書類の提出・承認状況に応じて自動でステータス更新
- **リアルタイム連動**: 一覧ページと詳細ページのステータスが完全同期
- **レスポンシブUI**: PC・タブレット・モバイル対応
- **書類承認フロー**: 個別書類の承認・差戻し → 全承認後に申請実行

## 🛠 技術スタック

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v3
- **UI Components**: shadcn/ui
- **Icons**: Lucide React
- **State Management**: React Context
- **Deployment**: Netlify (Static Site Generation)

## 📊 ステータス管理

### 学生ステータス
- **未提出**: 未提出書類がある状態
- **確認中**: 全書類提出済み、承認待ち
- **要修正**: 差戻しされた書類がある状態
- **提出済み**: 全書類承認済み、申請可能
- **申請中**: 入管への申請完了

### 書類承認フロー
1. 学生が書類を提出
2. 各書類を個別に「承認」または「差戻し」
3. 全書類承認完了で「申請」ボタンが有効化
4. 申請実行でステータスが「申請中」に更新

## 🎨 デザイン特徴

- **PC操作性重視**: 余白を最小化し、情報密度を向上
- **SpeedVisa風UI**: 実際のサービスを参考にした管理画面デザイン
- **色分けシステム**: 
  - 締切まで30日未満: 赤色（緊急）
  - 締切まで90日未満: 黄色（注意）
  - 締切まで90日以上: 緑色（安全）

## 📱 レスポンシブ対応

### デスクトップ (lg以上)
- 全カラム表示
- サイドバーフィルター
- 詳細な情報表示

### タブレット (md-lg)
- 重要カラムのみ表示
- コンパクトなレイアウト

### モバイル (sm以下)
- 必須情報のみ表示
- タッチ操作最適化
